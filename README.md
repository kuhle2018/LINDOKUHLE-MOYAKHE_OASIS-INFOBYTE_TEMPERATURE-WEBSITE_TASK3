# 🌡️ Climate Elegance – Temperature Converter

**Climate Elegance** is a beautifully designed temperature converter built by Lindokuhle Moyakhe. It allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin with precision and style. The project showcases elegant UI design, responsive layout, and interactive feedback—all wrapped in a luxury aesthetic.

---

## ✨ Features

- 🔄 Convert between Celsius, Fahrenheit, and Kelvin
- ✅ Input validation for accurate results
- 🌬️ Contextual feedback based on temperature range
- 🧠 Conversion history tracking
- 🎨 Elegant emerald-themed UI with gradient background
- 📱 Responsive design for desktop and mobile
- 📌 Sticky header with navigation links
- 📎 Footer with social links and author credit

---

## 📦 Technologies Used

- **HTML5** – Semantic structure
- **CSS3** – Custom styling, gradients, and layout
- **JavaScript** – Conversion logic and interactivity
- **Google Fonts** – `Playfair Display` and `Inter` for elegant typography

---

## 🚀 Getting Started

To run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/kuhle2018/temperature-converter.git
   ```

2. Navigate to the project folder:
   ```bash
   cd temperature-converter
   ```

3. Open `index.html` in your browser, or use Live Server in VS Code for real-time updates.
